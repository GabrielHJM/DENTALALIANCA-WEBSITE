# Dental Aliança — Excelência Odontológica

[![GitHub License](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/GabrielHJM/DENTALALIANCA-WEBSITE/blob/main/LICENSE)
[![Design Style](https://img.shields.io/badge/Design-Apple--Inspired-000000)](https://www.apple.com/design/)

Um ecossistema digital de alta performance desenvolvido para a **Dental Aliança**, referência em produtos odontológicos em Dourados, MS. Este projeto combina engenharia de software moderna com uma estética minimalista e fluida inspirada nos padrões de design da Apple.

---

## 💎 Filosofia do Projeto

O site foi construído sobre três pilares fundamentais:
1.  **Aestética Premium**: Interface limpa, tipografia Inter e uso estratégico de espaços em branco para transmitir confiança e qualidade.
2.  **Engenharia Fluida**: Implementação de um sistema de design responsivo baseado em algoritmos inteligentes (`CSS clamp`), garantindo que a experiência seja perfeita de 320px a 2560px.
3.  **Conversão Facilitada**: Arquitetura focada no cliente, com acesso direto a múltiplos canais de WhatsApp, localização precisa e marcas parceiras em destaque.

---

## 🛠️ Especificações Técnicas

- **Frontend Core**: HTML5 Semântico com foco em SEO.
- **Visual Engine**: CSS3 avançado (Custom Variables, Flexbox, Grid Layout).
- **Intelligent Logic**: Motor JavaScript Vanilla de alta performance para detecção de dispositivos e animações de revelação (Reveal Engine).
- **SEO & Search**: Dados estruturados JSON-LD (Schema.org) para indexação preferencial pelo Google.
- **Glassmorphism**: Navegação Blur-Saturated inspirada no macOS/iOS.

---

## 🚀 Funcionalidades Principais

- **Multi-Canal de Vendas**: Direcionamento direto para WhatsApp Vendas 1, Vendas 2 e Financeiro.
- **Localização Inteligente**: Integração com Google Maps para a R. Toshinobu Katayama, 443 - B.
- **Preloader Premium**: Experiência de entrada suave e profissional.
- **Catálogo de Categorias**: Organização moderna das principais áreas da odontologia.
